Introduction:

In this lab we tried to confirm Kirchhoff's Current Law and Kirchhoff's Voltage Law using a real circuit made from snap circuits. Kirchhoff's Voltage Law states that the sum of voltages around a loop is zero and Kirchhoff's Current Law says that the current entering a loop equals the current leaving it. These ideas come from the bigger concepts of conservation of Energy and Charge, and are very important for analysing circuits. My hypothesis was that our circuit board would verify this but with some minor errors.

Materials:

  • Snap Circuit Set
  • Assorted Resistors
  • Multimeters
  • 2 Batteries (3V total)
  • Recording method

Procedure:

Figure 1: Circuit Diagram

  1. Assemble the circuit as shown in the diagram above.
  2. Use resistors: R1 = 1 kΩ, R2 = 5.1 kΩ, R3 = 10 kΩ, R4 = 100 kΩ, R5 = 100 Ω.
  3. Connect the batteries in series to power the circuit.
  4. Use multimeters to measure voltages across and currents through each resistor.
  5. Record the voltage, current, and resistance values in a table.
  6. Use equations:
    Loop 1: V1 + V2 + V5 = Vs
    Loop 2: V3 + V4 = V2
    Node a: Is = I1
    Node b: I1 = I2 + I3
    Node c: I3 = I4
    Node d: I2 + I4 = I5
  7. Verify KVL and KCL using the recorded values.

Using the values from our one and only trial, we verified:

  • Loop 1: V1 + V2 + V5 ≈ Vs ⇒ 0.02 + 3.31 + 0.002 = 3.332 ≈ 3.36 V
  • Loop 2: V3 + V4 = 0.29 + 3.01 = 3.30 ≈ V2 = 3.31
  • Node b: I1 = I2 + I3 ⇒ 0.556 ≈ 0.53 + 0.025 = 0.555

Error Analysis:

Errors could be inaccuracies in multimeters, slight changes in battery voltage, and the resistance not measured in the components. To fix these errors, I could average readings over more trials or use more precise equipement.

Percent Error Calculation:

I calculated the percent error for loop 1 using the formula:

Percent Error = |Measured - Expected| / Expected × 100%
               = |3.36 V - 3.332 V| / 3.36 V × 100%
               ≈ 0.83%

The fact that there is low percentage of error supports that the laws have validity.

Conclusion:

This experiment confirmed both of Kirchhoff’s Laws using actual circuit measurements. The small inaccuracies were within a realistic error margin, supporting the idea that the laws apply to real-world circuits. Circuits have many uses in real life from basic home uses like light switches to the enigmatic source behind how computers work.
